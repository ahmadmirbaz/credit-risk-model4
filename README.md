# Credit Risk AI

A credit risk prediction system that estimates the probability of loan default using a logistic regression model and provides AI-generated explanations of risk factors.

## Features

• Logistic regression credit risk model  
• Financial feature engineering  
• AI-generated explanations of model predictions  
• REST API backend  
• Dashboard for viewing risk metrics  

## Tech Stack

Python  
Pandas  
NumPy  
scikit-learn  
PostgreSQL  
Express / Node.js  
OpenAI API  

## Run Locally

pnpm install  
cp .env.example .env  
pnpm --filter @workspace/api-server run dev
